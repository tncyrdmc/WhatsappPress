Wordpress plugin wich show a Whatsapp badge to start a conversation.

== Description ==

WhatsappPress will add a button to your pages making the visitors able to start a conversation with you using Whatsapp!  

== Installation ==

As simple as:

- upload the plugin on your site
- activate it
- change the settings
- profit?
